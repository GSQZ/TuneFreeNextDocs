# 常见问题

这里收集了用户在使用 TuneFreeNext 过程中遇到的常见问题及解答。

## ::material-symbols:download:: 下载与安装

::: details Android 安装时提示"禁止安装"怎么办？
请在手机设置中允许安装来自未知来源的应用，具体路径因手机品牌而异：
- **小米**：设置 → 隐私保护 → 特殊权限 → 安装未知应用
- **OPPO/vivo**：设置 → 安全 → 允许安装未知应用
- **华为**：设置 → 安全 → 更多安全设置 → 安装未知应用
:::

::: details Windows 安装时提示"无法验证发布者"怎么办？
点击"仍要运行"或"更多信息 → 仍要运行"即可继续安装。这是因为应用未进行代码签名，不影响正常使用。
:::

::: details 下载的音乐文件保存在哪里？
不同平台的下载路径：
- **Android**: `/storage/emulated/0/Download/Music/TuneFreeNext/`
- **Windows**: `%USERPROFILE%\Music\TuneFreeNext\`（通常是 `C:\Users\你的用户名\Music\TuneFreeNext\`）
- **macOS/Linux**: `~/Music/TuneFreeNext/`
:::

## ::material-symbols:play-circle:: 播放相关

::: details 为什么有些歌曲无法播放？
可能的原因：
1. **版权限制** - 该歌曲为付费/VIP 专属，暂时无法获取音源
2. **音源问题** - 当前音源无法解析，建议切换其他音源尝试
3. **网络问题** - 检查网络连接是否正常
4. **会员过期** - 部分高音质需要会员权限

建议：在设置中切换到其他音源重新搜索尝试。
:::

::: details 如何切换播放音质？
进入 **设置 → 播放音质**，可选择：
- 128k（流量省）
- 320k（标准）
- FLAC（无损）
- FLAC 24bit（Hi-Res）

注意：实际播放音质取决于音源是否提供对应质量的音频。
:::

::: details 如何使用定时关闭功能？
在全屏播放页面，点击顶栏的定时关闭图标，可选择：
- **定时分钟数**：10/15/30/45/60/90 分钟后关闭
- **播放完当前歌曲**：当前歌曲结束后关闭
- **播放指定歌曲数**：播放 N 首歌曲后关闭
:::

::: details 什么是无缝播放？
无缝播放会在当前歌曲开始播放后，自动在后台预加载下一首歌曲的播放地址。当切换到下一首时，可以立即开始播放，无需等待加载。

在 **设置 → 无缝播放** 中可以开关此功能。
:::

## ::material-symbols:playlist-add:: 歌单相关

::: details 如何导入在线歌单？
1. 复制歌单链接或歌单 ID
2. 进入 **歌单** 页面
3. 点击右上角的导入按钮
4. 粘贴链接，应用会自动识别来源
5. 点击确认后跳转到歌单详情
6. 点击收藏按钮将歌单保存到本地
:::

::: details 如何收藏歌曲？
有多种方式：
1. **长按歌曲** → 弹出菜单 → 点击「喜欢」
2. 在全屏播放页面点击**心形图标**
3. 在歌曲菜单中选择「添加到歌单」
:::

::: details 本地歌单数据存在哪里？
本地歌单数据存储在应用私有目录的 SQLite 数据库中，不会因为清除缓存而丢失。但卸载应用会导致数据丢失，建议使用**云同步**功能备份数据。
:::

## ::material-symbols:cloud-sync:: 账号与同步

::: details 如何注册账号？
1. 打开应用，进入登录页面
2. 点击「注册」
3. 输入邮箱地址
4. 点击「获取验证码」，到邮箱查收验证码
5. 输入验证码和密码，完成注册
:::

::: details 换设备后数据如何恢复？
1. 在新设备上登录同一账号
2. 进入 **设置 → 云同步**
3. 选择需要恢复的数据类型（歌单/收藏/设置）
4. 点击「从云端恢复」

注意：恢复会覆盖当前设备的本地数据。
:::

::: details 忘记密码怎么办？
1. 在登录页面点击「忘记密码」
2. 输入注册时使用的邮箱
3. 获取验证码并输入
4. 设置新密码
:::

::: details VIP 会员有什么特权？
VIP 会员可以：
- 解锁高音质音源（320k/FLAC/FLAC 24bit）
- 无限制播放和下载
- 支持开发者持续更新

充值方式：进入 **设置 → 账号信息 → 充值会员**
:::

## ::material-symbols:cached:: 缓存相关

::: details 如何清理缓存？
进入 **设置 → 缓存管理 → 清理缓存**，可以一键清除所有缓存的音频文件。

缓存设置选项：
- **缓存开关**：是否启用自动缓存
- **缓存大小**：最大缓存空间限制
- **过期时间**：缓存自动过期天数
:::

::: details 缓存和下载有什么区别？
| | 缓存 | 下载 |
|---|---|---|
| 存储位置 | 应用私有目录 | 公共音乐目录 |
| 是否可见 | 其他应用不可见 | 可在文件管理器中看到 |
| 是否会自动清理 | 是（超限/过期时） | 否 |
| 用途 | 减少重复加载 | 离线收藏 |
:::

## ::material-symbols:computer:: 桌面端相关

::: details 如何让 Windows 版关闭时最小化到托盘？
进入 **设置 → 关闭到托盘**，开启后点击关闭按钮会最小化到系统托盘而不是退出应用。

托盘操作：
- **单击**：显示/隐藏窗口
- **右键**：打开托盘菜单（播放控制、显示窗口、退出）
:::

::: details 桌面端如何调节音量？
在底部播放栏右侧，鼠标悬停在音量图标上会展开音量滑块，拖动即可调节。点击音量图标可以快速切换静音。
:::
