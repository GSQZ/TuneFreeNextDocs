# 欢迎使用 TuneFreeNext

感谢你选择 TuneFreeNext！这是一款简洁高效的跨平台音乐播放器，致力于为你带来自由、流畅的音乐体验。

## ::material-symbols:info:: 关于 TuneFreeNext

TuneFreeNext V3 基于 **Flutter** 全新开发，采用 **Material Design 3** 设计语言，一套代码运行于多个平台：

:::: card-grid
::: card title="已发布" icon="material-symbols:check-circle"

- **Android** - 手机、平板、横屏适配
- **Windows** - 桌面系统
- **车机** - 车载系统适配
- **TV** - 电视端适配

:::

::: card title="即将推出" icon="material-symbols:schedule"

- **iOS** - iPhone、iPad
- **macOS** - Mac 电脑
- **Linux** - Linux 桌面

:::
::::

- 关于我们的故事

  **起源与成长**
  
  自 2021 年 8 月首个版本 LuciferMusic 发布以来，我们历经溺溟音乐、简繁音乐、TuneFree 等多个版本的迭代，用户峰值曾突破百万级别。在这段旅程中，我也从高一学生成长为大二软件工程专业的学生。
  
  **技术演进**
  
  从最初的易语言，到 Vue、Python、Kotlin、Electron 的探索，再到如今使用 Flutter 实现真正的跨平台统一，TuneFree 不断尝试新的技术方向。这期间离不开众多优秀开源项目的启发与借鉴。
  
  **致谢**
  
  感谢每一位用户的支持与陪伴，TuneFreeNext 将继续为你带来更好的音乐体验！

:::
