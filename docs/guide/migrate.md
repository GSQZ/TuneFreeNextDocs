# 数据迁移

如果你之前使用过 TuneFree V1/V2 版本并开启了云同步，可以通过此页面将数据迁移到 V3 版本。

> [!WARNING]
> 
> V3 版本的云同步数据格式与 V1/V2 不兼容，需要进行数据迁移才能在 V3 中使用旧版数据。

## ::material-symbols:sync-alt:: 开始迁移

请先登录你的账号，然后进行数据迁移操作。

<MigrateApp />

## ::material-symbols:help:: 常见问题

::: details 什么情况下需要数据迁移？
如果你之前使用 TuneFree V1 或 V2 版本，并且开启了云同步功能保存过歌单或收藏，那么需要通过此页面将数据迁移到 V3 格式。
:::

::: details 迁移后旧数据会被删除吗？
不会。迁移操作只会将旧数据转换并保存为 V3 格式，原有的 V1/V2 数据会保留。
:::

::: details 迁移失败怎么办？
请检查网络连接，然后重试。如果多次失败，请 [加入 QQ 群](https://qm.qq.com/q/UmYH8A5CY8) 联系开发者协助处理。
:::
